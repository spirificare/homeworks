Будівельні теги (усі блочні):

  • DIV-тег без призначення: <div></div>
  • HEADER-шапка сайту/блоку: <header></header>
  • FOOTER-підвал сайту: <footer></footer>
  • MAIN-головна частина сторінки: <main></main>

Теги для контенту:

  • P-параграф тексту (блочний): <p></p>
  • A-посилання (рядковий): <a target="_blank" href="посилання">Назва посилання</a>
  • IMG-додавання зображення (в html рядковий): <img src="посилання" alt="Назва зображення">
  • BR-переревання рядка (МАЙЖЕ блочний): <br>
  • SPAN-виділення частини рядка (рядковий): <span></span>
  • MARK-виділяє/підсвічує частину рядка (рядковий): <mark></mark>
  • UL-маркований список (блочний): <ul> <li>Пункт</li> <li>Пункт</li> <li>Пункт</li> </ul>
  • OL-нумерований список (блочний): <ol> <li>Пункт</li> <li>Пункт</li> <li>Пункт</li> </ol>
  • H-заголовок (блочний): <h1>Я заголовок 1</h1> <h2>Я заголовок 2</h2> <h3>Я заголовок 3</h3> <h4>Я заголовок 4</h4> <h5>Я заголовок 5</h5> <h6>Я заголовок 6</h6>
